今天上午主要调研了`MarkDown For GitHub CheatSheet`

1. 标题

  1-6个`#`表示1-6级标题，最后一个`#`后加空格。

  # 标题1
  ## 标题2
  ### 标题3
  #### 标题4
  ##### 标题5
  ###### 标题6

2. 文本样式

  |样式|关键字|样例|
  |----|------|----|
  |加粗|`** abc **`|** abc **|
  |斜体|`_ abc _`|_ abc _|
  |删除线|`~~ abc ~~`|~~ abc ~~|
  |加粗&斜体|`** _ abc _ **`|** _ abc _ **|

3. 引用

  - 文本：`> abc` 
  
    > abc
  
  - 代码：
  
    - 内联： `abc`
	
	- 显示： 
	``` c++
	int a = 1;
	cout << a << endl;
	```
	
4. 链接

  - 行内链接
  
    [链接到百度](https://www.baidu.com "百度")
  
  - 参考链接
  
    [链接到百度][1]
  
  [1]: https://www.baidu.com

  1. abc
  2. abc
  3. abc
  
  *. abc
  *. abc
  *. abc
  
  - abc
  - abc
    - abc
	- abc
